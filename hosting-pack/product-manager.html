প্রোডাক্ট ম্যানেজার

বামে ক্যাটাগরি বেছে প্রোডাক্ট যোগ করুন — সেই পেজে + হোমেও অটো যাবে। শুধু ছবি, দাম, সাইজ, রঙ লিখুন।

বাল্ক ছবি →
ক্যাটাগরি = বামের তালিকা (লিংক আলাদা লিখতে হয় না):
  1. ① আবায়া বেছে প্রোডাক্ট যোগ → abaya পেজ + হোম
  2. ④ টু-পিস বেছে যোগ → premium-two-piece পেজ + হোম
  3. ⑤ এম্ব্রয়ডারি বেছে যোগ → embroidery পেজ + হোম (মেনুতে চাইলে enabled: true)
  4. প্রতিটি প্রোডাক্টে: ছবি, দাম, সাইজ, রঙ → সেভ → ফাইল রিপ্লেস → Netlify
