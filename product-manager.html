কীভাবে ব্যবহার করবেন:
  1. বামে ক্যাটাগরি বেছে নিন (① আবায়া … ⑧ হিজাব)
  2. প্রতিটি প্রোডাক্টের দাম, ফ্যাব্রিক, রঙ, সাইজ এডিট করুন
  3. আবায়া Type দাম: Full Set / Abaya Only আলাদা বক্সে লিখুন → 「সব প্রোডাক্টে এই দাম লাগান」
  4. + নতুন ক্যাটাগরি — মেনু নাম দিন, key ও HTML পেজ অটো তৈরি
  5. সেভ করুন → category-products.js + product-catalog-sections.js ডাউনলোড
  6. প্রজেক্ট ফোল্ডারে ফাইল রিপ্লেস → Ctrl+F5
