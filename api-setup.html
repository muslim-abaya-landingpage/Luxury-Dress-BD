⚡ doGet not found? — ৩০ সেকেন্ডে ঠিক

পুরনো Code.gs রেখে নতুন ফাইল যোগ করুন:

  1. Windows-এ API-দ্রুত-ঠিক.bat ডাবল-ক্লিক → Yes
  2. Editor-এ + → Script → নাম Fix-doGet → Ctrl+V → Save
  3. Deploy → Manage → Edit → New version → Deploy

সম্পূর্ণ সেটআপ (Sign Up সহ)

  1. apps-script-copy.html → পুরো কোড → Code.gs Replace
  2. Project Settings → AUTH_SECRET সেভ
  3. Deploy → New version
