# প্রোডাক্ট সহজ গাইড — ক্যাটাগরি অটো, হোমেও দেখাবে

## মূল কথা (একবার বুঝলেই হবে)

| আপনি যা করেন | কোথায় দেখাবে |
|--------------|----------------|
| বামে **① আবায়া** বেছে `+ প্রোডাক্ট যোগ` | **Abaya** পেজ + হোম (যদি চালু) |
| বামে **④ টু-পিস** বেছে যোগ | **Two-piece** পেজ + হোম |
| **⑤ টপস/কুর্তি** ইত্যাদি — এখনো যোগ না করলে | পেজ **খালি** থাকে, হোমেও **আসে না** |

**আলাদা লিংক লিখতে হয় না** — বামের ক্যাটাগরিতে যোগ করলেই সিরিয়ালে সেই পেজে বসে।  
`product-links-data.js` শুধু **ছবি** আপডেট করে; খালি ক্যাটাগরিতে লিংক দিলেও নতুন প্রোডাক্ট তৈরি হয় না।

প্রতি প্রোডাক্টে শুধু লিখুন: **ছবির URL, দাম, সাইজ, রঙ** (ইচ্ছামতো নাম, ফ্যাব্রিক)।

---

## ধাপ

1. **`প্রোডাক্ট-এডিট.bat`** চালান
2. **বামে ক্যাটাগরি** বেছে নিন (আবায়া / টু-পিস / এম্ব্রয়ডারি …)
3. **+ প্রোডাক্ট যোগ** → ছবি, দাম, সাইজ, রঙ
4. **সেভ করুন** → `category-products.js` + `product-links-data.js` ডাউনলোড
5. প্রজেক্টে রিপ্লেস → Netlify → `Ctrl+F5`

## মেনুতে ক্যাটাগরি চালু (Coming soon সরাতে)

হোমে প্রোডাক্ট থাকতে **enabled** লাগে না।  
শুধু **উপরের মেনু** থেকে ক্যাটাগরি খুলতে চাইলে `product-catalog-sections.js` এ:

```javascript
enabled: true   // সেই ক্যাটাগরির জন্য
```

## ভুল যেন না হয়

- আবায়ার ড্রেস **আবায়া** তালিকায় রাখুন — টু-পিস তালিকায় নয়।
- একই ছবি দুই ক্যাটাগরিতে কপি করলে হোমে একবারই দেখাবে (ডুপ্লিকেট এড়ায়)।
- HTML (`abaya.html`) বা checkout এডিট করতে হবে না।
